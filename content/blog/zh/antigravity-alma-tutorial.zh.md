## 实战教程：三步搞定

### Step 1：下载软件

根据自己的操作系统，分别在 GitHub 和软件官网下载上面两个软件：

- **Antigravity Tools**: 从 [GitHub Releases](https://github.com/lbjlaq/Antigravity-Manager) 下载
- **Alma**: 从 [官网](https://alma.now/) 下载

---

### Step 2：在 Antigravity Tools 登录 Gemini 账号

1. 打开 Antigravity Tools
2. 点击 **添加账号**，根据引导进行登录
3. 登录完成后，即可在 **API 反代** 中看到自己的 API 秘钥

---

### Step 3：在 Alma 配置对应的 API

1. 打开 Alma 的 **设置**
2. 进入 **提供商** 配置页面
3. 按照以下方式配置 API 信息
4. 点击 **Models** 旁边的 **Fetch** 获取模型

配置完成后，你可以看到获取到了三个模型。这三个模型底层使用的模型可以在 Antigravity Tools 中进行配置。

比如，你可以将 `gpt-3.5` 配置为使用 **Claude claude-opus-4-5-thinking** 进行对话。
