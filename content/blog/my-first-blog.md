## 关于这个博客

这个博客是使用 [Hugo](https://gohugo.io/) 静态网站生成器和 [Ladder](https://github.com/guangzhengli/hugo-theme-ladder) 主题搭建的。

Hugo 是一个用 Go 语言编写的快速、现代的静态网站生成器，它可以在一秒内渲染数千篇文章。

## 为什么选择静态博客

静态博客有很多优势：

1. **速度快** - 无需数据库查询，页面加载速度极快
2. **安全** - 没有后端，减少了被攻击的风险
3. **易于部署** - 可以免费托管在 GitHub Pages、Cloudflare Pages 等平台
4. **版本控制** - 所有内容都可以用 Git 管理
