## Tutorial: Three Easy Steps

### Step 1: Download

Download both software according to your operating system:

- **Antigravity Tools**: From [GitHub Releases](https://github.com/lbjlaq/Antigravity-Manager)
- **Alma**: From [Official Website](https://alma.now/)

---

### Step 2: Login to Gemini Account in Antigravity Tools

1. Open Antigravity Tools
2. Click **Add Account** and follow the guide to login
3. After login, you can see your API key in **API Proxy**

---

### Step 3: Configure API in Alma

1. Open Alma's **Settings**
2. Go to **Provider** configuration page
3. Configure the API information
4. Click **Fetch** next to **Models** to get models

After configuration, you can see three models available. The underlying models can be configured in Antigravity Tools.
